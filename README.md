# Python Projects Portfolio

A collection of Python projects built to demonstrate skills in APIs, web scraping, GUI development, and data handling. Each project focuses on real-world problem solving using Python.

# 📌 Projects Included

## 🪙 Crypto Price Tracker

A desktop app that fetches live cryptocurrency prices using a public API.

### Features
- Search any cryptocurrency
- View live USD price
- View 24-hour price change
- View market cap
- Save search history to JSON
- Handle invalid coins and API errors

### Tech Stack
- Python
- Tkinter
- Requests
- JSON
- CoinGecko API

---

## 🌤️ Weather App

A GUI app that fetches real-time weather data for any city.

### Features
- Search weather by city
- Display temperature and conditions
- Show humidity and wind speed
- Handle invalid city inputs
- Store search history

### Tech Stack
- Python
- Tkinter
- Requests
- JSON
- Weather API

---

## 🐙 GitHub User Finder

A desktop application that fetches GitHub user data using the GitHub API.

### Features
- Search GitHub users
- Display profile information (bio, followers, repos, etc.)
- Save searched users
- Handle invalid usernames and API errors

### Tech Stack
- Python
- Tkinter
- Requests
- JSON
- GitHub API

---

## 📚 Personal Library Manager

A web scraping project that collects and analyzes book data from an online bookstore.

### Features
- Scrape multiple pages of books
- Search books by title
- Filter books by price
- Find cheapest and most expensive books
- Calculate average price
- Save data to JSON
- GUI-based interface

### Tech Stack
- Python
- BeautifulSoup
- Requests
- Tkinter
- JSON
